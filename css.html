O que é CSS?

O que é

CSS é uma linguagem usada para descrever a apresentação de documentos HTML, incluindo aspectos como layout, cores e fontes. Ele permite separar o conteúdo da estrutura visual, facilitando a manutenção e o design de páginas web.

Como funciona

CSS funciona aplicando estilos a elementos HTML através de seletores e propriedades, permitindo personalizar a aparência de páginas web com regras de estilo. As regras são aplicadas em cascata, o que significa que estilos podem ser herdados ou sobrescritos com base em especificidade e ordem.

Conceitos Básicas

Propriedades e Valores

Propriedades

Características dos elementos HTML que você pode estilizar, como color, font-size, margin, etc.

Valores

Valores atribuídos às propriedades, definindo o estilo específico. Por exemplo, em color: red;, color é a propriedade e red é o valor.

Cascata e Especificidade

Cascata

Determina como os estilos são aplicados com base na ordem e origem das regras. Estilos definidos mais tarde ou com maior especificidade substituem os anteriores.

Especificidade

A prioridade das regras CSS é definida pela especificidade dos seletores. Seletores de ID têm a maior prioridade, seguidos pelos seletores de classe e atributos, que possuem uma prioridade intermediária. Por fim, seletores de tag têm a menor prioridade.
