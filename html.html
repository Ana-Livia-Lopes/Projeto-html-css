O que é HTML?

O que é

HTML é uma linguagem de marcação fundamental para a criação e formatação de sites na web, evoluindo para ser usada com CSS e JavaScript no desenvolvimento de aplicativos web dinâmicos.

Como funciona

Um site médio inclui diversas páginas HTML separadas que consistem em elementos formados por tags e atributos, renderizados por navegadores para exibição aos usuários.

Tags HTML Básicas

Parágrafos

A tag < p > em HTML é usada para definir um parágrafo de texto, sendo cercada por uma tag de abertura < p > e uma de fechamento < /p > para indicar onde o parágrafo começa e termina. criando um bloco de texto separado em uma página web. Essa tag é essencial para a organização do conteúdo textual, permitindo uma leitura clara e estruturada para os usuários.

Listas

  1. para produzir uma lista você pode utilizar as tags < ul >, < ol >, e < li >.
  2. < ul >: Define uma lista não ordenada, onde os itens não têm uma ordem específica.
  3. < ol >: Define uma lista ordenada, onde os itens são sequenciados numericamente ou alfabeticamente.
  4. < li >: Define um item de lista e pode ser usado dentro de < ul > ou < ol > para representar cada item na lista

Tabelas

Passo Descrição
1. Definir a tabela Use a tag < table > para iniciar a tabela.
2. Adicionar cabeçalhos Use a tag < thead > para criar um cabeçalho e a tag < tr > para a linha, com tags < th > para cada coluna.
3. Adicionar linhas Use a tag < tbody > para o corpo da tabela e a tag < tr > para cada linha, com tags < td > para as células.
4. Adicionar rodapé Opcionalmente, use a tag < tfoot > para adicionar um rodapé com < tr > e < td > para as células.

Imagens

Para colocar uma imagem em HTML, use a tag < img >, especificando o caminho do arquivo de imagem no atributo src. Por exemplo: < img src="imagem.jpg" alt="Descrição da imagem" >. O atributo alt fornece uma descrição alternativa para a imagem, importante para acessibilidade.

Links

Para colocar um link em HTML, use a tag < a >, que é a tag de âncora. Coloque o URL de destino dentro do atributo href da tag. Por exemplo, < a href= "https://www.exemplo.com" > Texto do Link < /a > cria um link que leva o usuário ao endereço especificado. O "Texto do Link" é o texto clicável que será exibido na página. Para abrir o link em uma nova aba, adicione o atributo target="_blank", como em < a href="https://www.exemplo.com" target="_blank" > Texto do Link.
